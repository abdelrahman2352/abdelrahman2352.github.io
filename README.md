# أدوات دُرّة — أدوات PDF والصور

موقع مجاني لأدوات PDF والصور يعمل 100% داخل المتصفح. مستضاف على GitHub Pages بالدومين `tools.durrahsystem.tech`.

## أدوات متاحة
- دمج PDF
- تقسيم PDF
- حذف صفحات PDF
- تدوير PDF
- تحويل الصور إلى PDF
- PDF إلى صور
- ضغط الصور
- تحويل صيغ الصور

## المتطلبات
- Node.js 18+
- npm 8+

## التشغيل المحلي
npm install
npm run dev

## البناء للإنتاج
npm run build

## النشر على GitHub Pages
يتم النشر تلقائيًا عند الـ push. يجب أن يكون مجلد `dist` هو مجلد النشر في إعدادات GitHub Pages.
